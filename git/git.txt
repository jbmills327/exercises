New Repo
1. create repo in github and copy the url they give
2. cd to root dir of repo
3. git init
4. git remote add origin <url>
5. git add -A
6. git commit -m "your message"
7. git push origin master

Typical used to push up work will be git add, git  commit and git push
1. git add -A
2. git commit -m "your message"
3. git push origin master

To get new files in a repo 
1. cd to root of repo
2. git pull origin master
